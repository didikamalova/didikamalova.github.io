Behavior Cloning from Multiple Demonstrators

  • Tools: Python, PyTorch
  • Project URL: GitHub

This project focuses on providing a method that helps BC policies improve monotonically with respect to the number of demonstrators by carrying out an experiment to condition a policy based on a demonstrator. By conditioning the BC agent on whoâ€™s collecting the data, we can work with data under the same distribution. We recover conditional distribution which no longer violates the unimodality requirement. The experiment setup involves the robotic arm and a gripper placed on a table with multiple objects. 5 different demonstrations decreasing in performance (increasing noise) and 11 policies, involving 10 conditioned & unconditioned policies each trained on the single demonstrator data and 1 unconditioned policy trained on the combined data, were trained.
