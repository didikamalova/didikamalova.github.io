Can You Judge a Book by Its Cover?

  • Tools: Python, Image & Title Processing
  • Project URL: GitHub

This project investigates the task of using book cover images and title text to classify the book into one of 30 genres. We employed three methods for this task: (a) an image recognition model for book covers, (b) a language processing model for book titles, and (c) a linear combination of outputs from the two previous models. The output of all models is a 30-vector that signifies the model’s prediction for each genre.
